### TMUX

Terminal MultipleXer

To use tmux with plugins make sure you install `bash` with homebrew

- macOS default path : `/bin/bash`
- brew installed path : `/opt/homebrew/bin/bash`

#### Prerequisites

```sh
brew install bash
```

#### Install TMUX

```sh
brew install tmux
```

### Install TPM

Tmux Plugin Manager

```sh
git clone https://git.github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
