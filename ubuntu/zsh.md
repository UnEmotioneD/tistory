# Z-Shell

- Larger ecosystem compare to BASH

## Install ZSH

```bash
sudo apt install zsh
```

---

## Change Shell

```bash
chsh -s $(which zsh)
```

- reboot

## Plugins

- install oh-my-zsh
- use powerlevel10k

- autosuggestion
- history-substring-search
- fast-syntax-highlighting

- clone with git
- source it from `~/.zshrc`
