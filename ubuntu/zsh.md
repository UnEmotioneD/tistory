# Z-Shell

- Larger ecosystem compare to BASH

## Install ZSH

```bash
sudo apt install zsh
```

---

## Change Shell

- theme
  - omz
    - powerlevel10k

```bash
chsh -s $(which zsh)
```

- then restart

## Plugins

- autosuggestion
- history-substring-search
- fast-syntax-highlighting

- clone with git
- source it from `~/.zshrc`
