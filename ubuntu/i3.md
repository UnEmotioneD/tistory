## Install Packages

- `i3`: the window manager
  - `i3lock`: lock screen
  - `i3status`: statusbar
  - `dmenu`: app launcher
- `alacritty`: i3 sensible terminal
- `feh`: wallpaper
- `picom`: compsiter(fixes screen tearing)

- `polybar`: better statusbar
- `rofi`: better app launcher

```bash
sudo apt install -y i3 alacritty feh picom polybar rofi
```

## Input

- mouse sensitivity

create config file

```bash
sudo nvim /etc/X11/xorg.conf.d/40-mouse.conf
```

just put this in there and reboot

10cm from edge to edge on QHD monitor

```conf
Section "InputClass"
    Identifier "Disable accel and slow mouse"
    MatchIsPointer "on"
    Driver "libinput"
    Option "AccelProfile" "flat"
    Option "AccelSpeed" "-0.13"
EndSection
```

- keyrepeat
  - delay: 250ms
  - repeat speed: 25ms

```conf
exec_always --no-startup-id xset r rate 250 25
```

## Key Binding

- Add this line to `config`

```text
focus_wrapping no
```

- windows key to mod
- mod + h, j, k, l for changing focus
- mod + y, u, i, o, p, n, m for workspaces
- screen shot

### Default Key

- mod + r: resize
- mod + s: stack mod
- mod + v: next window will open in vertical position

- mod + shift + r: reload i3 config
- mod + shift + q: quit app

- tiling/floating
- parent/child

## Dmenu

- if you have installed an app but doesn't show up on demnu refersh by removing dmenu cache

```bash
rm ~/.cash/dmenu_run
```

- in i3 config file instead of `dmenu_run` `i3-dmenu-desktop` only shows desktop app

```conf
bindsym Mod1+space exec --no-startup-id i3-dmenu-desktop
```

## Status bar

- left: workspaces
- right: input source, wifi, bluetooth, speaker, cpu, temperature, battery, time

- copy i3status config from `/etc/i3status.conf` to `~/.config/i3status/config`

```bash
mkdir -p ~/.config/i3status
cp /etc/i3status.conf ~/.config/i3status/config
```

## Assign App to Workspace

Use `xprop` to get app's class name

`assign` command only opens the app on specific workspace

with the following way you start the program and focus will also change

```conf
for_window [class="Google-chrome"] move to workspace $ws2, focus
for_window [class="Alacritty"]     move to workspace $ws3, focus
for_window [class="Code"]          move to workspace $ws4, focus
```

## Monitor Setting

show information with xrandr

```bash
xrandr -q
```

monitor, res, fps

```bash
xrandr --output DP-4 --mode 2560x1440 --rate 239.97
```

dual monitor use `--left-of`

```bash
xrandr --output DP-4 --mode 2560x1440 --rate 239.97 --output DP-5 --mode 1920x1080 --rate 60.00 --left-of DP-4
```

add the command above with `exec_always --no-startup-id` at `i3/config`

- Assign workspace `7` or `y` to secondary monitor
