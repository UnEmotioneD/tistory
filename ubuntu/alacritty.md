## Install

### Add PPA

```sh
sudo add-apt-repository ppa:aslatter/ppa
```

### Update APT

```sh
sudo apt update
```

### Install Alacritty

```sh
sudo apt install alacritty
```

### Check Version

```sh
alacritty --version
```

---

## Uninstall

### Remove Alacritty

```sh
sudo apt remove alacritty
sudo apt purge alacritty
```

### Remove PPA

```sh
sudo add-apt-repository --remove ppa:aslatter/ppa
sudo apt update
```

Check uninstalled

```sh
which alacritty
```
