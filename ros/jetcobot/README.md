# Jetcobot

Packages to install to use `Jetcobot`

## PIP Packages

Inside a `venv` update `pip` itself

```sh
pip install --upgrade pip

# or
python3 -m pip install --upgrade pip
```

---

## For Jetcobot

To control it

```sh
pip install pymycobot --upgrade
pip install jupyter opencv-python
```

---

## For Flask

To use webcam service

### Both on Server and Client

```sh
pip install flask flask-socketio opencv-python python-socketio
```
