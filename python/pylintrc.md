# Python Linter Remote Control

Pylint configuration

## Suppress Warnings

```dosini
[MESSAGES CONTROL]
disable =
    missing-docstring,
    too-few-public-methods,
    wrong-import-order,
```

`missing-docstring` includes

- `missing-module-docstring`
- `missing-class-docstring`
- `missing-function-docstring`
