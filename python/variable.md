# Variable naming

| Convention           | Format           | Use Case                     |
| -------------------- | ---------------- | ---------------------------- |
| floatcase            | foldername       | Directory                    |
| PascalCase           | MyClassName      | Class                        |
| SCREAMING_SNAKE_CASE | MY_CONSTANT_NAME | constant variable            |
| snake_case           | my_variable_name | variable & function & module |
